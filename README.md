# Proyecto de Robot Móvil en Entorno de Construcción

Simulación usando un robot diferencial Pioneer P3DX en CoppeliaSim.

## Características

- Control de robot diferencial
- Implementación de algoritmos de generación de rutas BiTRRT (Bidirectional Transition-based RRT)
- Transporte de objetos
- Localización del robot
- Evitación de obstáculos mediante computación reactiva

## Tecnologías utilizadas

- Python
- CoppeliaSim
- Lua
